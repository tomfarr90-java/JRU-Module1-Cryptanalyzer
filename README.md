Криптоанализатор 

Консольное приложение на Java для шифрования, расшифровки и взлома текста с использованием алгоритма Цезаря.

Программа поддерживает:
1. "Шифр Цезаря" - шифрование и дешифрование по ключу. 
2. "Brute Force" - автоподбор ключа по на основе пунктуации и популярных союзов.
3. "Статистический анализ" - взлом текста путем анализа частотности самых популярных символов (пробел, о, е, а);

Технологии: 
Java 25 (для старых версий дописать в класс Main public static void main(String[] args))
